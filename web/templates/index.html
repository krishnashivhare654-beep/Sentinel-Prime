ABOUT SENTINEL PRIME

Sentinel Prime is a comprehensive cybersecurity suite developed by Krishna Shivhare at VIT Bhopal University.

  • ShadowTrace Integration: Advanced IP geolocation and threat mapping.
  • Vault-X: Military-grade AES-256 encryption core for secure data storage.
  • Network Intelligence: Real-time traffic analysis and device discovery.

Tech Stack: Python (Flask), SQLite, Chart.js, Vercel Serverless.

SYSTEM DOCUMENTATION (HELP)

1. Encryption: Enter full file path and secret key in Vault-X. Click ENCRYPT to secure the file.

2. Network Scan: Click INITIATE_SCAN to perform an ARP scan on the local subnet (10.106.204.0/24).

3. Traffic Analysis: The doughnut chart shows protocol distribution (TCP/UDP/ICMP).

4. Reports: Generate a PDF summary of the last 24h activity via DOWNLOAD_REPORT.
